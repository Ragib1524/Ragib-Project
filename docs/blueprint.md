# **App Name**: ResumeAI Pro

## Core Features:

- User Authentication: Secure signup, login, and logout functionality using JWT.
- Resume Builder: Step-by-step form for entering personal details, experience, education, skills, and projects with real-time preview.
- AI Resume Enhancement: AI-powered tool to rewrite bullet points, improve grammar, optimize for ATS, and suggest improvements based on the user's role.
- Role-Specific Suggestions: AI provides tailored recommendations based on user-selected job role (e.g., software engineer, designer).
- Resume Templates: Selection of modern, ATS-friendly resume templates.
- PDF Export: Download resumes as pixel-perfect PDFs.
- Dashboard: Manage (create, edit, duplicate, delete) user resumes.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) to convey professionalism and trust.
- Background color: Light gray (#F0F4F7) for a clean and modern look.
- Accent color: Teal (#3EB489) for interactive elements and highlights.
- Body text: 'Inter' (sans-serif) for a clean, readable, and modern feel.
- Headline text: 'Space Grotesk' (sans-serif) provides a tech-focused aesthetic, and complements the Inter body text.
- Simple, consistent icons to represent resume sections and actions.
- Clean, SaaS-style layout with clear visual hierarchy and ample white space.
- Subtle transitions and loading animations to enhance UX.
